# Rules for Writing References (Referências)

## Purpose
The References section lists ALL — and ONLY — the sources that were cited in the body of the text. Works that were read but not cited do NOT belong here (they go in a separate "Bibliography" section, if required).

## Core Rule
**Only cite what you referenced. Only reference what you cited.**

## Formatting Rules (ABNT NBR 6023)
- Title: "REFERÊNCIAS" — centered, bold, all caps
- Listed in **ascending alphabetical order** by the first element (usually the author's last name)
- Single spacing within each reference; one blank line between references
- Font type and size same as the body text (e.g., Times New Roman 12)
- No numbering or bullet points

## Citation Types and Formats

### Book (Single Author)
```
LAST NAME, First Name. Title of the book: subtitle. Edition. City: Publisher, Year.
```
Example:
```
ROESCH, S. M. A. Projeto de estágio e de pesquisa em administração. São Paulo: Atlas, 2009.
```

### Book (Multiple Authors)
```
LAST NAME, First Name; LAST NAME, First Name. Title. City: Publisher, Year.
```

### Book Chapter / Part of a Book
```
LAST NAME, First Name. Title of the chapter. In: LAST NAME, First Name (Ed.). Title of the book. City: Publisher, Year. p. xx-xx.
```

### Journal Article
```
LAST NAME, First Name. Title of the article. Journal Name, City, volume, number, p. xx-xx, Month Year.
```

### Academic Work (Thesis, Dissertation, Monograph)
```
LAST NAME, First Name. Title of the work. Year. Number of pages. Type of work (Degree level) — Institution, City, Year.
```

### Website / Online Document
```
LAST NAME, First Name. Title of the document. Available at: <URL>. Accessed on: DD Mon. YEAR.
```
Example:
```
PEDAGOGIA em foco. Glossário. Available at: <http://www.pedagogiaemfoco.pro.br/met10.htm>. Accessed on: 26 Jan. 2016.
```

### Brazilian Technical Standards (ABNT)
```
ABNT – ASSOCIAÇÃO BRASILEIRA DE NORMAS TÉCNICAS. NBR XXXX: Title. Rio de Janeiro, Year.
```
Example:
```
NBR 6028. Resumos. Rio de Janeiro: ABNT, 2003.
```

## Special Cases

### Same Author, Multiple Works
- List chronologically (oldest to newest)
- Use underline or dashes for repeated author name after first entry:
```
GIL, A. C. Como elaborar projetos de pesquisa. São Paulo: Atlas, 1991.
__________. Métodos e técnicas de pesquisa social. São Paulo: Atlas, 2009.
```

### Works with No Identified Author
- Begin with the title (first word capitalized, rest in lower case unless proper noun)

### Works Used via apud (Citation of Citation)
- List BOTH works: the original and the work that cited it
- The one actually read (the citing work) must appear in References
- The original (not read) may be listed in a footnote or omitted from References

## Checklist Before Submitting
- [ ] Every source cited in the text appears in References
- [ ] No source appears in References that wasn't cited in the text
- [ ] Alphabetical order is correct
- [ ] All required bibliographic data is present for each entry
- [ ] Formatting is consistent (ABNT style)
- [ ] Title "REFERÊNCIAS" is bold, centered, all caps

## Common Mistakes to Avoid
- Do NOT include sources you read but did not cite
- Do NOT forget to include all data (publisher, year, city, page) for each entry
- Do NOT use numbered lists or bullet points
- Do NOT mix citation styles (APA, Vancouver, etc.) — use ABNT throughout
- Do NOT forget online sources need an access date
